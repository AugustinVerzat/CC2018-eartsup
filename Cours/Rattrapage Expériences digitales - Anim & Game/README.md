## Rattrapage Expériences digitales

### Sujet : Créer une page web présentant un projet de creative coding.

Pour ce sujet vous devrez choisir sur le site https://devart.withgoogle.com/fr#/ un projet qui vous inspire. A partir de ce projet vous devrez créer une page web (HTML & CSS).

####Contenu de la page 
Votre page devras contenir les parties et informations suivantes : 
 - Description 
  - Le nom de l'oeuvre
  - Le nom de l'auteur
  - Le lien vers la page DevArt
  - Votre nom
 - Une présentation de l'artiste / studio / société
 - Une présentation du projet (texte & image)
  - Pourquoi (Quel est le but du projet)
  - Comment (Les technologies utilisées, ce que vous comprenez, les étapes de production et la manière de travailler)
 - Pourquoi avez-vous choisi ce projet?
 - En quoi le code pourrai vous être utile dans votre futur métier?

#### Forme du rendu

Votre contenu devras être mis en forme dans une page web. Pour ce faire vous devrez utiliser les langages HTML et CSS que nous avons en cours.
Ainsi votre contenu devras être formater avec les balises HTML adéquates (Titre, paragraphe...) et chaque parties devrons être séparer à l'aide d'une structure logique.
Votre page devras avoir un design attrayant et permettre de distinguer les différentes parties et niveau d'information.

Votre rendu devras être envoyé sur le serveur GitHub du cours dans un dossier rattrapage qui aura cette nomenclature :
- Rattrapage
 - index.html
 - style.css
 - image
   - image1.jpg
   - image2.jpg
   -  ...
