Run and Collect

-Date : 11/04/2018
-Dev : Cailbourdin Tristan
-Titre : Run and Collect
-Exercice : À partir des éléments vus en cours, réaliser un mini-jeu autour de la grille. Ce dernier pourra s'inspirer de l'exemple du cours (the maze) ou se baser sur un autre gameplay (jeu de la taupe, enigma, puzzle...). Pour cela vous utiliserez l'ensemble des connaissances apprises en cours et les mettrez en application dans votre jeu.
-Consigne Jeu : Déplacements du joueur avec les touches flechées. Le joueur doit se déplacer pour collecter les points verts, mais il doit éviter les points rouges sinon Game Over
